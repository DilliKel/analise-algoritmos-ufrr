## logaritmos

| Propriedade | Fórmula |
|---|---|
| Mudança de base | log_a b = log_c b / log_c a |
| Produto | log_a (b · c) = log_a b + log_a c |
| Quociente | log_a (b / c) = log_a b - log_a c |
| Potência | log_a b^c = c · log_a b |
| Logaritmo Natural | ln ab = ln a + ln b |
| Logaritmo Natural de e | ln e = 1 |
| Logaritmo Natural de 1 | ln 1 = 0 |
| Mudança de base para logaritmos naturais | log_a b = ln b / ln a |
