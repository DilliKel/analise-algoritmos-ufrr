## revisão de logaritmos

Um logaritmo é uma maneira de descobrir qual o expoente que precisamos usar em um número (chamado de base) para obter outro número. Por exemplo, se quisermos saber qual o expoente que precisamos usar na base 2 para obter 8, podemos escrever isso como log₂8 = 3, já que 2³ = 8. Em outras palavras, o logaritmo é o inverso da exponenciação.

### logaritmos naturais

### propriedades

#### propriedade de multiplicação de logaritmos

A propriedade de produto de logaritmos é uma das propriedades mais importantes dos logaritmos e é usada para simplificar expressões logarítmicas que envolvem produtos. A propriedade afirma que o logaritmo de um produto é igual à soma dos logaritmos dos fatores, ou seja:

log_a(b * c) = log_a(b) + log_a(c)

| Propriedade | Fórmula |
|---|---|
| Mudança de base | log_a b = log_c b / log_c a |
| Produto | log_a (b · c) = log_a b + log_a c |
| Quociente | log_a (b / c) = log_a b - log_a c |
| Potência | log_a b^c = c · log_a b |
| Logaritmo Natural | ln ab = ln a + ln b |
| Logaritmo Natural de e | ln e = 1 |
| Logaritmo Natural de 1 | ln 1 = 0 |
| Mudança de base para logaritmos naturais | log_a b = ln b / ln a |
